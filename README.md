# Stock Scraper

A simple Go application that scrapes stock data from yahoo finance and provides useful insights for analysis, including the calculation of Discounted Cash Flow (DCF) and Weighted Average Cost of Capital (WACC).

## Features

- Scrape real-time stock data (e.g., price, volume, market cap).
- Supports multiple stock symbols.
- Calculate Discounted Cash Flow (DCF) for stock valuation.
- Calculate Weighted Average Cost of Capital (WACC) for financial analysis.
- Store or display scraped data for further analysis.
